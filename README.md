# gerenciamento-de-ativos

## 🎯 Sobre 

Este repositório possui imagens referentes ao TCC estudo de caso aplicado em uma organização de TI de médio porte em cidade XXXX, focando na implementação de boas práticas de gerenciamento de ativos baseadas no framework ITIL 4. O trabalho aborda desde a análise inicial até a validação dos processos implementados, com ênfase na melhoria da eficiência operacional e na otimização do ciclo de vida dos ativos de TI.

## 📂 Estrutura do Repositório:

### /Tabela-comparativa-objetivos

Conteúdo: Imagem com a análise comparativa dos objetivos esperados versus objetivos alcançados nos fluxogramas implementados.

Finalidade: Demonstrar o alinhamento entre as metas teóricas estabelecidas e os resultados práticos obtidos durante o estudo de caso.

### /Visões
Conteúdo: Conjunto de imagens representando as três perspectivas principais do projeto:

Visão de Implementação - 
Visão de Implantação - 
Visão de Validação - 
Finalidade: Apresentar visualmente as diferentes dimensões do processo de adoção das práticas ITIL 4 na organização estudada.

### /Fluxogramas
Conteúdo: Diagramas de processos fundamentais para o gerenciamento de ativos de TI:

Fluxograma de Triagem - Processo de classificação e priorização inicial de ativos
Fluxograma de Rastreamento - Monitoramento e localização de ativos no ciclo de vida
Fluxograma de Relacionamento - Interconexões e dependências entre ativos
Finalidade: Documentar visualmente os processos operacionais desenvolvidos e implementados conforme o framework ITIL 4.

### /Guia
Conteúdo: Documento-guia completo desenvolvido durante o trabalho para auxiliar a organização no processo de implantação das práticas de gerenciamento de ativos.

Finalidade: Apresenta um modelo prático de gerenciamento de ativos orientado pelas boas práticas do ITIL 4, com funcionalidades para:

* Orientar e capacitar profissionais da organização
* Documentar processos e procedimentos
* Definir padrões operacionais e metas mensuráveis
* Servir como referência para sustentabilidade da implementação

### /Inventário
Conteúdo: Imagem do dashboard inicial do sistema de inventário de ativos de TI implementado na organização do estudo de caso.

Finalidade: Demonstrar visualmente a interface e funcionalidades da solução prática desenvolvida, mostrando métricas, status e visão consolidada dos ativos gerenciados.

### /Itil-images
Conteúdo: Imagens de referência conceitual extraídas da documentação oficial ITIL Foundation da Axelos:

* Cadeia de Valor do Serviço - Representação visual dos seis componentes da cadeia de valor.
* Sete Princípios Norteadores - Ilustração dos princípios fundamentais do ITIL 4.
* Finalidade: Fornecer base conceitual visual para o embasamento teórico do trabalho, conectando teoria à prática implementada.

🔗 Conexão entre Componentes
Os elementos deste repositório formam um ecossistema completo:

Fundação teórica (Itil-images) → Modelo prático (Guia) → Processos operacionais (Fluxogramas).

Implementação (Visões) → Ferramenta prática (Inventário) → Avaliação de resultados (Tabela-comparativa).

## 📄 Utilização dos Recursos

Os materiais deste repositório são referenciados no documento principal do TCC e servem como:

* Suporte visual para explicação de processos complexos;
* Documentação gráfica dos resultados obtidos;
* Material complementar para apresentações e defesas;
* Guia prático para replicação em outros contextos organizacionais.
